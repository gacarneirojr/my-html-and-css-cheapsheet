HTML & CSS Cheat Sheet

HTML Forms

Tag Name Description
< form > Form Defines a form
< select > Select Creates a scrolling menu. Size sets the number of menu items visible before user needs to scroll.
< option > Option Sets off each menu item
< textarea > Text Area Creates a text box area. Columns set the width; rows set the height.
< input type="checkbox" > Checkbox Creates a checkbox
< input type="checkbox" name=? value=? checked > Pre checked checkbox Creates a checkbox which is pre-checked.
< input type="radio" name=? value=? > Radio button Creates a radio button.
< input type="radio" name=? value=? checked > Pre checked radio button Creates a radio button which is pre-checked.
< input type="text" name=? size=? > One-line text area Creates a one-line text area. Size sets length, in characters.
< input type="submit" value=? > Submit Button Creates a submit button. Value sets the text in the submit button.
< input type="image" name=? src=? border=? alt=? > Submit Button with Image Creates a submit button using an image.
< input type="reset" > Reset button Creates a reset button

CSS Selectors

Tag Name Description
.class Class Selects all elements with class="intro"
.class1.class2 Classes Selects the element with id=""
* All Selects all elements
element Element Selects all < p > elements
element.class Element -> Class Selects all < p > elements with class="intro"
[attribute] Attribute Selects all elements with a 'samples' attribute.
:active Ative Selects the active link
::after After Insert something after the content of each < p > element
:enabled Enabled Selects every enabled < input > element
:focus Focus Selects the input element which has focus
:hover Hover Selects links on mouse over.
:link Link Selects all unvisited links
